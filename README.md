# News Agent

## What It Is

News Agent is an interactive web dashboard for following daily international news, technology updates, market trends, and selected stocks or crypto assets in one place.

It collects live news and market information from public sources and organizes it into a simple dashboard for daily tracking.

## Tech Stack

- React
- Vite
- Node.js
- JavaScript
- RSS feeds
- Yahoo Finance chart data
- Render
